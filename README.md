# Mind vs Coin — React Conversion

This repo is a React conversion of the original `index.html` game. Files were split into components, original CSS was preserved, and audio/confetti/AI logic were ported to React hooks.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- The original audio/image files are included as empty placeholders in `public/assets/`. Replace them with the originals from `c:\Users\BISWAJIT\OneDrive\Desktop\web disigning\game` to restore sounds and avatar.
- Main app entry: [src/App.jsx](src/App.jsx)
- Styles: [src/index.css](src/index.css)
- Components: [src/components](src/components)
